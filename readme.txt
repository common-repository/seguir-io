# Adicione um widget do Instagram em seu site. O widget irá exibir seu nome de usuário, número de seguidores e um botão para seguir seu Instagram.
# Add a widget with your Instagram into your site. The widget displays your username, number of followers and button to follow your Instagram.

== Installation ==

=== Português ===
Instale e ative o plugin.

O plugin será adicionado na seção de widgets do seu WordPress. A partir de lá, você preenche seu usuário do Instagram e assim pode inserir o widget no rodapé, sidebars, etc.

Você também pode usar o seguinte shortcode para inserir o widget em qualquer página:

`[seguirio-widget username="yourUsername" width="100%" theme="light"]`

Para mais informações sobre os parâmetros que podem ser usados, por favor visite https://seguir.io e leia a documentação através do link no menu.

=== English ===
Install and activate the plugin.

The plugin will be added to the widgets section of WordPress. From there, you can type your Instagram username and insert the widget in your footers, sidebars, etc.

You can also use the following shortcode to insert the widget on any page:

`[seguirio-widget username="yourUsername" width="100%" theme="light"]`

For more information on the parameters that can be used, please visit https://seguir.io and check the documentation link on menu.
